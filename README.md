## Run Locally

Clone the project

```bash
  git clone https://github.com/HowlVFX/password-generator.git
```

Go to the project directory

```bash
  cd password-generator
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  streamlit run streamlit_app.py
```
